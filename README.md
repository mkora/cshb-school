### Glossary for Beginners

 
.NET - a free, cross-platform, open source developer platform for building many different types of applications

CLR - Common Language Runtime (part of .NET) - a managed execution environment that transforms source code into a form of bytecode known as CIL. Take cares of memory, security, exceptions.

CIL - Common Intermediate Language

BCL - Base Class Library - Base classes

FCL - Framework Class Library

DLL - Dynamic-link Library is Microsoft's implementation of the shared library concept in the Microsoft Windows and OS/2 operating systems

Common Type System - https://docs.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/types-and-variables

.NET Documentation https://docs.microsoft.com/en-us/dotnet/index?view=netframework-4.7.2

Nuget - The package manager for .NET: https://www.nuget.org/

LINQ - .NET Language-Integrated Query: https://msdn.microsoft.com/en-us/library/bb308959.aspx and Introduction to LINQ https://docs.microsoft.com/en-us/dotnet/csharp/programming-guide/concepts/linq/introduction-to-linq

SSMS - SQLs Server Management Studio

WPF - Windows Presentation Foundation

UWP - Universal Windows Platform

GDI+ - Windows Graphics Device Interface

XAML - Extensible Application Markup Language

XBAP - XAML browser application

WCF - Windows Communication Foundation - a framework for building service-oriented applications

IIS - Internet Information Services for Windows® Server - a flexible, secure and manageable Web server for hosting anything on the Web

.NET Core CLI - a CLI for developing cross platform websites and services on Linux, macOS, and Windows

### .NET Glossary

#### Platforms:

.NET Core - cross-platform .NET for websites, servers, and console apps on Windows, Linux, and macOS

.NET Framework - .NET implementation for websites, services, desktop apps on Windows

Xamarin/Mono - .NET implementation for running apps on mobile OSs

#### .NET Languages:

- C# - object-oriented, and type-safe p.language

- F# - cross-platform, open-source, functional p.language

- Visual Basic - simple syntax p.language for building type-safe, object-oriented apps

#### Application models:
 
- Web (building web apps and services for Windows, Linux, macOS, and Docker)
 
- Mobile (building native mobile apps for iOS, Android, and Windows)

- Desktop (building desktop apps for Windows and macOS)
 
- Gaming (building 2D/3D games for desktops, phones, and consoles)
 
- Machine Learning & AI
 
- Internet of Things (building IoT apps, with native support for the Raspberry Pi and other single-board computers)

#### .NET Desktop Apps

- WPF (Windows Presentation Foundation) - a UI framework for building visually compelling Windows desktop apps

- WinForms (Windows Forms) - a UI framework, with drag-and-drop designers, for building Windows desktop apps

- UWP (Universal Windows Platform) extends the .NET to enable development for any Windows 10 device—PC, tablet, phone, Xbox, HoloLens, Surface Hub, and Windows 10 IoT Core

- Xamarin - a set of tools and libraries for building cross-platform apps (native apps for macOS, iOS, Android, and Windows on a shared .NET codebase). Note: Shared an average of 75% of app code across all platforms

.NET Mobile Apps

#### ASP.NET:

- ASP.NET offers three frameworks for creating web applications:

	1. Web Forms (Win Forms & WPF: drag-and-drop, event-driven model)
	
	2. ASP.NET MVC (for creating SPA)

	3. ASP.NET Web Pages (classic ASP - HTML Markup)

- ASP.NET Web API is a framework that makes it easy to build HTTP services

- ASP.NET SignalR is a library that makes developing real-time web functionality easier (SignalR allows bi-directional communication between server and client and supports Web Sockets)

- ASP.NET SPA helps you build applications that include significant client-side interactions using HTML 5, CSS 3 and JavaScript

- WebHooks is a lightweight HTTP pattern providing a simple pub/sub model for wiring together Web APIs and SaaS services

- Xamarin
